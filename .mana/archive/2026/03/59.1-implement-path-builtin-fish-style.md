## Task
Implement a `path` builtin with subcommands for path manipulation. Replaces basename/dirname/realpath with a single modern command.

## Behavior
- `path basename [PATH...]` — get filename component
- `path dirname [PATH...]` — get directory component
- `path extension [PATH...]` — get file extension
- `path stem [PATH...]` — filename without extension (same as `path change-extension ''`)
- `path resolve [PATH...]` — resolve to absolute path (realpath)
- `path normalize [PATH...]` — normalize path (remove ./ and ../)
- `path join PARTS...` — join path components
- `path is [-f|-d|-l|-e] [PATH...]` — test path type (file/dir/link/exists)
- All subcommands read from stdin if no PATH args

## Implementation
1. Create `src/builtins/path_cmd.rs` with subcommand dispatch
2. Use `std::path::Path` methods: `.file_name()`, `.parent()`, `.extension()`, `.file_stem()`
3. Use `std::fs::canonicalize()` for resolve
4. Register in mod.rs

- `src/builtins/path_cmd.rs` (create)
- `src/builtins/mod.rs` (modify)
