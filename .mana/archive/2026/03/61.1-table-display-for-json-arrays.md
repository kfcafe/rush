## Task
When `--json` output is a JSON array of objects, render it as an aligned, colored table instead of raw JSON. This makes Rush's structured data story visually competitive with nushell.

## Example
```
$ ls --json | table
NAME          SIZE    MODIFIED
Cargo.toml    2.1K    2025-01-15
src/          -       2025-01-20
README.md     8.4K    2025-01-18
```

## Implementation
1. Create `src/output/table.rs` — detect JSON arrays of objects, render as aligned columns
2. Auto-detect terminal width for column sizing
3. Color headers, alternate row shading optional
4. Pipe to `table` builtin, or auto-detect when stdout is a TTY

## Files
- `src/output/table.rs` (create)
- `src/output/mod.rs` (modify)
- `src/builtins/mod.rs` (modify — register `table` builtin if standalone command)
