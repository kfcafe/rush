## Task
Add fish-style autosuggestions: as you type, show grey ghost text from history that completes the current input. Press right-arrow or end to accept.

## Implementation
Reedline has built-in hinter support:
1. Use `reedline::DefaultHinter` or implement a custom hinter using Rush's history backend
2. Configure it in the reedline editor setup (likely in `src/main.rs` where the reedline Editor is created)
3. Set the hint style to grey/dim text: `DefaultHinter::default().with_style(Style::new().fg(Color::DarkGray))`

## Context
Reedline's hinter API:
```rust
let hinter = Box::new(DefaultHinter::default().with_style(Style::new().fg(Color::DarkGray)));
let mut line_editor = Reedline::create().with_hinter(hinter);
```
