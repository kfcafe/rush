## Task
Implement `mv` as a fast in-process builtin with undo tracking.

## Behavior
- `mv SRC DEST` — rename/move file or directory
- `mv -f SRC DEST` — force overwrite
- `mv -n SRC DEST` — don't overwrite existing
- `mv SRC1 SRC2... DIR` — move multiple into directory
- Cross-device fallback: copy + delete when `fs::rename` fails with EXDEV

## Implementation
1. Create `src/builtins/mv.rs`
2. Use `std::fs::rename` (fast same-device), fall back to copy+delete for cross-device
3. Integrate with undo manager
4. Register in mod.rs, add help

- `src/builtins/mv.rs` (create)
- `src/builtins/mod.rs` (modify)
