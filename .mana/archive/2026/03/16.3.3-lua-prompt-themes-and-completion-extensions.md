## Task

Allow Lua scripts to customize the prompt and add completion sources.

## Prompt themes
Lua scripts register prompt segments that are called before each prompt render:

```lua
rush.register_prompt("kubernetes", function()
    local ctx = rush.exec("kubectl config current-context 2>/dev/null")
    if ctx ~= "" then
        return { text = "☸ " .. ctx, color = "cyan" }
    end
end)
```

The RushPrompt in main.rs collects all registered prompt segments and renders them in order.

## Completion extensions
```lua
rush.register_completion("docker", function(prefix, context)
    if context.command == "docker" and context.arg_index == 1 then
        return {"run", "build", "push", "pull", "ps", "images", "logs"}
    end
end)
```
