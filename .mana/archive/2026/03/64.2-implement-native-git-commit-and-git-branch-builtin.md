## Task
Implement `git commit` and `git branch` via native git2 bindings.

## Behavior — git commit
- `git commit -m "message"` — commit staged changes
- `git commit -a -m "message"` — stage all tracked + commit
- `git commit --amend` — amend last commit (optional, can defer)
- Use `repo.signature()`, `repo.find_tree()`, `repo.commit()`

## Behavior — git branch
- `git branch` — list branches
- `git branch NAME` — create branch
- `git branch -d NAME` — delete branch
- `git branch -m OLD NEW` — rename branch
- `git branch --json` — JSON output
