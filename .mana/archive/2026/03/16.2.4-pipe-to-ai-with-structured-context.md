## Task

Implement the `|?` operator that's already in the parser AST. Pipes command output (structured or text) to the AI with an optional prompt.

## Examples
```bash
cat error.log |? "what's wrong here?"
git diff |? "review this change"
ls -la |? "which files are largest?"
```

## Behavior
1. Execute left side, capture output (structured or text)
2. Send to AI with user's prompt (default: "Analyze this output")
3. AI responds — read-only analysis, no tool usage needed
