## Task
Implement three small utility builtins inspired by fish.

## Behavior
### status
- `status` — print shell status summary
- `status is-interactive` — exit 0 if interactive, 1 otherwise
- `status is-login` — exit 0 if login shell
- `status filename` — print current script name
- `status line-number` — print current line number

### contains
- `contains KEY VALUES...` — exit 0 if KEY is in VALUES, 1 otherwise
- `contains -i KEY VALUES...` — also print the 1-based index
- Useful in conditionals: `if contains "$x" a b c; then ...`

### count
- `count ARGS...` — print number of arguments
- `count $array` — print array length
- Exit status 0 if count > 0, 1 if count == 0

## Implementation
1. Create `src/builtins/status_cmd.rs` — needs runtime access for is-interactive etc.
2. Create `src/builtins/contains.rs` — simple linear search
3. Create `src/builtins/count.rs` — trivially `args.len()`
4. Register all in mod.rs
