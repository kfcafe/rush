## Task

Replace reqwest (212 transitive deps, async runtime, OpenSSL) with ureq (minimal sync HTTP client, ~30 deps). reqwest is only used in the `fetch` builtin (src/builtins/fetch.rs).

## Approach
1. Add `ureq = { version = "3", features = ["json"] }` to Cargo.toml
2. Remove `reqwest` from Cargo.toml
3. Rewrite src/builtins/fetch.rs to use ureq API
4. ureq is sync by default — no async runtime needed

## API mapping
- `reqwest::blocking::Client` → `ureq::Agent`
- `client.get(url).send()` → `ureq::get(url).call()`
- `response.text()` → `response.into_body().read_to_string()`
- `response.json()` → `response.into_body().read_json()`
- Headers, methods, timeouts all have ureq equivalents

## Files
- Cargo.toml (modify — swap reqwest for ureq)
- src/builtins/fetch.rs (modify — rewrite to use ureq)
