## Task
Implement `chmod` and `chown` as builtins.

## Behavior — chmod
- `chmod MODE FILE...` — set permissions (octal: `chmod 755 file`, symbolic: `chmod u+x file`)
- `chmod -R MODE DIR` — recursive
- Parse both octal (755) and symbolic (u+x,g-w,o=r) modes

## Behavior — chown
- `chown OWNER[:GROUP] FILE...` — change ownership
- `chown -R OWNER[:GROUP] DIR` — recursive
- Use `nix::unistd::chown` (nix crate already in deps)
