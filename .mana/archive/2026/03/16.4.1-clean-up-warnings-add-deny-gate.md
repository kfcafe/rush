## Task

Fix all ~220 compiler warnings and add `#![deny(warnings)]` to src/lib.rs to prevent them from accumulating again.

## Common warning types to fix
- Unused imports → remove them
- Unused variables → prefix with `_` or remove
- Lifetime elision suggestions → apply (e.g., `Cow<str>` → `Cow<'_, str>`)
- Dead code → remove or `#[allow(dead_code)]` if intentionally reserved
- Missing docs on public items → add or `#[allow(missing_docs)]`

## Approach
1. Run `cargo fix --bin rush --allow-dirty` to auto-fix what it can
2. Manually fix remaining warnings
3. Add `#![deny(warnings)]` to src/lib.rs
4. Verify zero warnings with `cargo check`
