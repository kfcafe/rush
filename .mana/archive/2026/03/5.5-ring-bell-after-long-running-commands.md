## Task
After a command finishes that took longer than a configurable threshold, emit a BEL character (`\x07`) to stderr. This causes the terminal to bounce the dock icon or show a notification badge, alerting the user that their long build/test/download is done.

## Behavior
- Default threshold: 10 seconds (configurable via `RUSH_BELL_THRESHOLD` environment variable, in seconds; set to 0 to disable)
- Only in interactive mode (the REPL loop)
- Emit BEL to stderr (not stdout) so it doesn't interfere with piped output
- The bell fires for ALL commands over the threshold, regardless of success/failure

## Steps
1. In `src/main.rs`, in `run_interactive_with_reedline`, before the main loop:
   - Read `RUSH_BELL_THRESHOLD` env var, parse to u64, default to 10. Store as a `Duration`.
2. In the main loop, wrap command execution with timing:
   - Record `Instant::now()` before `execute_line`.
   - After execution completes, check `start.elapsed() >= threshold`.
   - If over threshold and threshold is not zero, write `\x07` to stderr.
3. This applies to both the normal execution path and the intent/abbreviation paths.
4. Add a helper function to keep the loop body clean: `fn maybe_bell(elapsed: Duration, threshold: Duration)` in main.rs or terminal.rs.

## Context
The REPL loop already has this structure:
```rust
// Normal command execution
match execute_line(exec_line, &mut executor) {
    Ok(result) => {
        // ... print output ...
        last_exit_code = Some(result.exit_code);
    }
    Err(e) => {
        eprintln!("Error: {}", e);
        last_exit_code = Some(1);
    }
}
```

Just wrap the `execute_line` call with `let start = Instant::now();` and check after.

## Files
- src/terminal.rs (modify — add `pub fn bell_if_long(elapsed: Duration, threshold: Duration)`)
- src/main.rs (modify — add timing around command execution, read threshold from env)

## Don't
- Don't add any sound/notification beyond BEL — terminal handles that
- Don't emit bell in non-interactive mode
- Don't make this complicated — it's a 3-line function and a few lines in the loop
