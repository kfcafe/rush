## Task
Implement `math` builtin with floating-point support, and `$RANDOM` variable.

## Behavior
- `math "1 + 2"` → 3
- `math "3.14 * 2"` → 6.28 (floating point!)
- `math "sin(pi)"` → 0
- `math "ceil(3.2)"` → 4
- `math "floor(3.8)"` → 3
- `math "round(3.5)"` → 4
- `math "abs(-5)"` → 5
- `math "sqrt(16)"` → 4
- `math "2 ^ 10"` → 1024
- `$RANDOM` — returns random integer 0-32767 (POSIX convention)

## Implementation
1. Create `src/builtins/math_cmd.rs`
2. Extend the existing arithmetic evaluator (`src/arithmetic.rs`) or write a new one with f64 support
3. Add function evaluation: sin, cos, tan, ceil, floor, round, abs, sqrt, log, ln, pi, e
4. Register `$RANDOM` as a special variable in runtime that generates on access
5. Register in mod.rs

## Files
- `src/builtins/math_cmd.rs` (create)
- `src/arithmetic.rs` (read — existing integer evaluator, possibly extend)
- `src/runtime/mod.rs` (modify — add $RANDOM special variable)
- `src/builtins/mod.rs` (modify)
