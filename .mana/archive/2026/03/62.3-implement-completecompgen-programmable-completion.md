## Task
Implement programmable completion — let users define custom tab completions for commands. This is essential for the plugin/extension story.

## Behavior
- `complete -c COMMAND -s SHORT -l LONG -a ARGS -d DESC` — add completion spec
- `complete -e -c COMMAND` — erase completions for command
- `compgen -W WORDLIST` — generate completions from word list
- `compgen -f` — generate file completions
- `compgen -d` — generate directory completions
- `compgen -c` — generate command completions

## Implementation
1. Create `src/builtins/complete_cmd.rs`
2. Store completion specs in runtime
3. Integrate with existing completion engine (`src/completion/`)
4. Register in mod.rs

## Files
- `src/builtins/complete_cmd.rs` (create)
- `src/completion/` (modify — read user-defined specs)
- `src/builtins/mod.rs` (modify)
