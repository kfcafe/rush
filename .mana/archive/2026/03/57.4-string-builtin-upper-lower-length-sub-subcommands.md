## Task
Add upper, lower, length, sub (substring), and collect subcommands to `string`.

## Behavior
- `string upper [STRING...]` — convert to uppercase
- `string lower [STRING...]` — convert to lowercase
- `string length [STRING...]` — print character count of each string
- `string length -q` — quiet mode, just set exit status
- `string sub -s START -l LENGTH [STRING...]` — extract substring (1-based)
- `string sub -s START -e END` — substring with end index
- `string collect [STRING...]` — collect into single output (no word splitting)
