## Task
Implement abbreviations (fish's `abbr`). Unlike aliases, abbreviations expand inline as you type — you see the full command before pressing enter. This is better for learning and transparency.

## Behavior
- `abbr -a NAME EXPANSION` — add abbreviation
- `abbr -e NAME` — erase abbreviation
- `abbr` — list all abbreviations
- When user types an abbreviation and presses space/enter, it expands inline
- Store persistently in `~/.config/rush/abbreviations`

- `src/builtins/abbr.rs` (create)
