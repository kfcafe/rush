## Task
Add `match` and `replace` subcommands to the `string` builtin.

## Behavior
- `string match PATTERN [STRING...]` — glob match, print matching strings
- `string match -r PATTERN [STRING...]` — regex match (PCRE-style)
- `string match -e PATTERN [STRING...]` — print entire string including non-matched parts
- `string match -i PATTERN [STRING...]` — case insensitive
- `string match -v PATTERN [STRING...]` — invert (print non-matching)
- `string replace PATTERN REPLACE [STRING...]` — literal replace
- `string replace -r PATTERN REPLACE [STRING...]` — regex replace
- `string replace -a PATTERN REPLACE [STRING...]` — replace all occurrences

## Implementation
1. Create `src/builtins/string_cmd/match_cmd.rs`
2. Create `src/builtins/string_cmd/replace.rs`
3. Use the `regex` crate (or `grep-regex` already in deps) for -r mode
4. Wire into the dispatch in `string_cmd/mod.rs`
