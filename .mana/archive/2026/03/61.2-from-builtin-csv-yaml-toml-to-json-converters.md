## Task
Implement a `from` builtin that converts structured text formats into JSON for use with Rush's `json_*` builtins.

## Behavior
- `from csv [FILE]` — parse CSV into JSON array of objects (headers from first row)
- `from yaml [FILE]` — parse YAML into JSON
- `from toml [FILE]` — parse TOML into JSON
- All read from stdin if no file given: `cat data.csv | from csv | json_query '.[]'`

## Implementation
1. Create `src/builtins/from.rs` with subcommand dispatch
2. For CSV: use a simple parser or add `csv` crate
3. For YAML/TOML: `serde_yaml` / `toml` crates (check before adding — ask if adding deps)
4. Output as JSON via `serde_json`
5. Register in mod.rs

## Files
- `src/builtins/from.rs` (create)
- `src/builtins/mod.rs` (modify)
- `Cargo.toml` (may need new deps — csv, serde_yaml, toml)
