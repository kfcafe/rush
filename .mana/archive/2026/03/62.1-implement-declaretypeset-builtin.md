## Task
Implement `declare`/`typeset` — variable attribute declaration, commonly found in bash scripts.

## Behavior
- `declare VAR=VALUE` — declare variable (like assignment)
- `declare -i VAR` — integer attribute
- `declare -r VAR` — readonly (like `readonly`)
- `declare -x VAR` — export (like `export`)
- `declare -a VAR` — indexed array
- `declare -A VAR` — associative array
- `declare -p VAR` — print declaration
- `declare` — list all variables with attributes
- `typeset` — alias for `declare`
