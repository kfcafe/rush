## Task
Implement `git add` via native git2 bindings — no fork/exec.

## Behavior
- `git add FILE...` — stage files
- `git add .` — stage all changes
- `git add -A` — stage all (including deletions)
- `git add -p` — interactive/patch mode (defer to external git for this)

## Implementation
1. Create `src/builtins/git_add.rs`
2. Use `git2::Repository::open()`, then `repo.index()`, `index.add_path()`, `index.write()`
3. Wire into `builtin_git` match in `src/builtins/mod.rs`

## Files
- `src/builtins/git_add.rs` (create)
- `src/builtins/mod.rs` (modify — add match arm in builtin_git)
