## Task
Three common pipeline/text-processing builtins.

## Behavior — tee
- `tee [FILE...]` — copy stdin to stdout AND to each FILE
- `tee -a [FILE...]` — append instead of overwrite

## Behavior — cut
- `cut -d DELIM -f FIELDS [FILE...]` — extract fields (e.g., `cut -d: -f1 /etc/passwd`)
- `cut -c CHARS [FILE...]` — extract character positions
- `cut -b BYTES [FILE...]` — extract byte positions

## Behavior — tr
- `tr SET1 SET2` — translate characters (e.g., `tr a-z A-Z`)
- `tr -d SET1` — delete characters in SET1
- `tr -s SET1` — squeeze repeated characters
- Character classes: `[:upper:]`, `[:lower:]`, `[:digit:]`, `[:space:]`
