## Task
Implement `pushd`, `popd`, and `dirs` builtins. The runtime already has full directory stack support.

## Context — existing runtime API (src/runtime/mod.rs:427)
```rust
pub fn push_dir(&mut self, dir: PathBuf)
pub fn pop_dir(&mut self) -> Option<PathBuf>
pub fn get_dir_stack(&self) -> &[PathBuf]
pub fn clear_dir_stack(&mut self)
```

## Behavior
- `pushd DIR` — push current dir onto stack, cd to DIR
- `pushd` (no args) — swap top two dirs
- `popd` — pop top dir, cd to it
- `dirs` — print the stack
- `dirs -c` — clear the stack
- `dirs -v` — print with index numbers

## Implementation
1. Create `src/builtins/dir_stack.rs` with all three builtins
2. Register all three in mod.rs
3. The cd logic should reuse the existing `builtin_cd` pattern (update runtime cwd + process cwd)
4. Add tests
