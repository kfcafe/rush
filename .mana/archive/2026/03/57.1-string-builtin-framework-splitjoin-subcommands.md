## Task
Create the `string` builtin framework with subcommand dispatch, plus the first two subcommands: `split` and `join`. Follow fish shell's `string` design: https://fishshell.com/docs/current/cmds/string.html

## Behavior
- `string split SEP [STRING...]` — split each string on SEP, one result per line
- `string split -m MAX SEP [STRING...]` — limit number of splits
- `string split -r SEP [STRING...]` — split from right
- `string split -n SEP [STRING...]` — exclude empty results
- `string join SEP [STRING...]` — join strings with SEP
- `string join0 [STRING...]` — join with NUL byte
- All subcommands read from stdin if no STRING args given

## Implementation
1. Create `src/builtins/string_cmd/mod.rs` — subcommand dispatch
2. Create `src/builtins/string_cmd/split.rs` — split/split0
3. Create `src/builtins/string_cmd/join.rs` — join/join0
4. Register `string` in `src/builtins/mod.rs`
5. Support stdin via `execute_with_stdin` pattern
6. Tests for each subcommand
