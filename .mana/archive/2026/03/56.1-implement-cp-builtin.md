## Task
Implement `cp` as a fast in-process builtin. Very common command — eliminating fork/exec overhead benefits AI agent and CI workloads.

## Behavior
- `cp SRC DEST` — copy file
- `cp -r SRC DEST` — recursive copy for directories
- `cp -p SRC DEST` — preserve permissions/timestamps
- `cp -f SRC DEST` — force overwrite
- `cp -n SRC DEST` — don't overwrite existing
- `cp SRC1 SRC2... DIR` — copy multiple files into directory
- Track via undo manager for undo support

## Implementation
1. Create `src/builtins/cp.rs`
2. Use `std::fs::copy` for single files, recursive walk for directories
3. Handle cross-device copies (same as mv fallback)
4. Register in mod.rs, add help entry
5. Integrate with undo tracking: `runtime.undo_manager()`
