## Task
Add trim, pad, repeat, and escape subcommands to `string`.

## Behavior
- `string trim [STRING...]` — remove leading/trailing whitespace
- `string trim -l` / `string trim -r` — left/right only
- `string trim -c CHARS` — trim specific characters
- `string pad -w WIDTH [STRING...]` — pad to width with spaces
- `string pad -r` — pad on right instead of left
- `string pad -c CHAR` — pad with specific char
- `string repeat -n COUNT [STRING...]` — repeat N times
- `string repeat -m MAX` — max output chars
- `string escape [STRING...]` — escape for shell safety
- `string escape --style=url` — URL encoding
- `string unescape [STRING...]` — reverse of escape
