## Task
Implement `ulimit` — set/get resource limits for the shell process. Cannot be external.

## Behavior
- `ulimit -a` — display all limits
- `ulimit -n [N]` — get/set open file limit
- `ulimit -u [N]` — get/set max user processes
- `ulimit -v [N]` — get/set virtual memory limit
- `ulimit -s [N]` — get/set stack size
- `ulimit -c [N]` — get/set core dump size
- `ulimit -S` — soft limit (default)
- `ulimit -H` — hard limit

## Implementation
1. Create `src/builtins/ulimit.rs`
2. Use `libc::getrlimit()` and `libc::setrlimit()` with the appropriate `RLIMIT_*` constants
3. Map flags to resource types: -n → RLIMIT_NOFILE, -u → RLIMIT_NPROC, etc.
4. Handle "unlimited" display when value == RLIM_INFINITY
5. Register in mod.rs, add help entry
