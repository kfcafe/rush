## Task
Implement the POSIX `umask` builtin. This CANNOT be an external command — it must modify the shell process's file creation mask.

## Behavior
- `umask` — print current mask in octal (e.g., `0022`)
- `umask MASK` — set mask (octal, e.g., `umask 077`)
- `umask -S` — print in symbolic form (e.g., `u=rwx,g=rx,o=rx`)
- `umask -S MASK` — set using symbolic form

## Implementation
1. Create `src/builtins/umask.rs`
2. Use `libc::umask()` to get/set the mask
3. Register in `src/builtins/mod.rs`: add `mod umask;` and `m.insert("umask", umask::builtin_umask);`
4. Add tests for: get current, set octal, set symbolic, -S flag

## Context
Follow the pattern in existing builtins like `src/builtins/set.rs`. The builtin signature is:
```rust
pub fn builtin_umask(args: &[String], runtime: &mut Runtime) -> Result<ExecutionResult>
```

## Files
- `src/builtins/umask.rs` (create)
- `src/builtins/mod.rs` (modify — add mod + register)
