## Task
Implement `shopt` (shell options) and `mapfile`/`readarray` (read lines into array).

## Behavior — shopt
- `shopt -s OPTION` — enable option (e.g., `globstar`, `nullglob`, `dotglob`)
- `shopt -u OPTION` — disable option
- `shopt OPTION` — query status (exit 0 if on, 1 if off)
- `shopt` — list all options and their status
- Key options: `globstar` (recursive glob **), `nullglob` (no matches → empty), `dotglob` (glob matches dotfiles)

## Behavior — mapfile
- `mapfile ARRAY < file` — read lines of file into array variable
- `mapfile -t ARRAY` — trim trailing newlines from each line
- `mapfile -n COUNT` — read at most COUNT lines
- `readarray` — alias for `mapfile`
