## Task
Auto-load functions from `~/.config/rush/functions/` — when a command is not found as a builtin or external, check if a `.rush` file exists with that name and load it. This is how fish's 960+ completion scripts work without slowing startup.

## Behavior
- On startup, scan `~/.config/rush/functions/` for function names (don't execute yet)
- When an unknown command is invoked, check if a matching function file exists
- If found, source the file (which defines the function), then call the function
- Cache loaded functions in memory
