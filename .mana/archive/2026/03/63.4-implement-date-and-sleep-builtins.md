## Task
Implement `date` and `sleep` builtins — common in scripts, trivial to implement.

## Behavior — date
- `date` — print current date/time
- `date +FORMAT` — strftime-style formatting (e.g., `date +%Y-%m-%d`)
- `date -d STRING` — parse and display a date string
- `date -u` — UTC time
- Rush already depends on `chrono` — use it.

## Behavior — sleep
- `sleep SECONDS` — sleep for N seconds
- `sleep 0.5` — fractional seconds
- `sleep 1m` / `sleep 1h` — duration suffixes (optional, nice-to-have)
- Use `std::thread::sleep`
