## Task
Implement `touch` builtin — create files or update timestamps without forking.

## Behavior
- `touch FILE...` — create files if they don't exist, update mtime/atime if they do
- `touch -c FILE` — don't create, only update existing files
- `touch -t TIMESTAMP FILE` — set specific time (format: [[CC]YY]MMDDhhmm[.ss])
- `touch -a FILE` — change only atime
- `touch -m FILE` — change only mtime

## Implementation
1. Create `src/builtins/touch.rs`
2. For creation: `std::fs::OpenOptions::new().create(true).write(true).open(path)`
3. For timestamp update: use `libc::utimensat` or `filetime` crate (check if already available, prefer libc since it's already a dep)
4. Register in `src/builtins/mod.rs`
5. Add undo tracking via `runtime.undo_manager()` for newly created files
6. Tests: create new, update existing, -c flag, multiple files

- `src/builtins/touch.rs` (create)
- `src/builtins/mod.rs` (modify — add mod + register)
