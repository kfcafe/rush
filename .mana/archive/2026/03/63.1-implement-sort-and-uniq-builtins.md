## Task
Implement `sort` and `uniq` as builtins — pipeline staples.

## Behavior — sort
- `sort [FILE...]` — sort lines alphabetically
- `sort -n` — numeric sort
- `sort -r` — reverse
- `sort -u` — unique (deduplicate)
- `sort -k N` — sort by field N
- `sort -t SEP` — field separator
- Reads from stdin if no file

## Behavior — uniq
- `uniq [FILE]` — remove adjacent duplicate lines
- `uniq -c` — prefix lines with occurrence count
- `uniq -d` — only print duplicates
- `uniq -u` — only print unique lines
- Reads from stdin if no file
