Add new types to src/parser/ast.rs and add match arms everywhere Argument is matched.

1. Add before the Argument enum:
```rust
#[derive(Debug, Clone, PartialEq, Serialize, Deserialize)]
pub enum ArgumentPart {
    Literal(String),
    Variable(String),
    BracedVariable(String),
    CommandSubstitution(String),
}
```

2. Add two variants to Argument:
```rust
SingleQuoted(String),
DoubleQuoted(Vec<ArgumentPart>),
```

3. Add match arms in EVERY file that matches on Argument so it compiles.
   Run: rg "Argument::" src/executor/ src/parser/commands.rs
   Key functions to update:
   - src/executor/expansion.rs: resolve_argument (~line 394), resolve_argument_static (~876), expand_args_to_strings (~620)
   - src/executor/mod.rs: resolve_argument (~2232), resolve_argument_static (~2818), plus ~5 inline matches (~315, ~355, ~2102, ~2121, ~2461, ~2470)
   - src/executor/pipeline.rs: resolve_argument (~546), expand globs (~623)
   - src/executor/commands.rs: inline matches (~22)

   For SingleQuoted: return string as-is (no expansion). For DoubleQuoted:
   expand each part and concatenate. Use this pattern:
   ```rust
   Argument::SingleQuoted(s) => Ok(s.clone()),
   Argument::DoubleQuoted(parts) => {
       let mut result = String::new();
       for part in parts {
           match part {
               ArgumentPart::Literal(s) => result.push_str(s),
               ArgumentPart::Variable(v) => {
                   result.push_str(&self.resolve_argument(&Argument::Variable(v.clone()))
                       .unwrap_or_default());
               }
               ArgumentPart::BracedVariable(v) => {
                   result.push_str(&self.resolve_argument(&Argument::BracedVariable(v.clone()))
                       .unwrap_or_default());
               }
               ArgumentPart::CommandSubstitution(c) => {
                   result.push_str(&self.resolve_argument(&Argument::CommandSubstitution(c.clone()))
                       .unwrap_or_default());
               }
           }
       }
       Ok(result)
   }
   ```
   For static resolve functions (no self), use the same pattern but call the static version.
   For inline matches that extract a string (like `Argument::Literal(s) | Argument::Variable(s) | ...`),
   add SingleQuoted(s) to the chain. For DoubleQuoted, join parts as literal strings.

4. Do NOT change how the parser produces arguments yet. Just add types and compile.
